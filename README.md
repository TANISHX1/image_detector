# image_detector
 GUI app that will be able to extract feature from image and also able to classify various type of image using python


## setup
```sh
# Create a virtual environment (recommended)
python -m venv image_classifier_env

# Activate it
# Windows:
image_classifier_env\Scripts\activate
# Mac/Linux:
source image_classifier_env/bin/activate

# Install required packages
pip install tensorflow
pip install pillow
pip install numpy
pip install matplotlib
pip install requests
pip install opencv-python
```
